# 프로젝트 기획서: Zen-Pop (초등학생용 힐링 영단어 게임)

## 1. 프로젝트 개요
- **목표:** 초등학생이 영어 단어 암기에 대한 스트레스를 벗어나, 게임처럼 즐겁게 몰입할 수 있는 모바일/태블릿용 영단어 학습 앱 개발.
- **핵심 컨셉 (Zen-Pop):** 공부 앱의 딱딱함을 버리고, 힐링(Healing) 요소와 톡톡 튀는 팝(Pop)한 감성을 결합한 RPG 형태의 게이미피케이션 적용.

## 2. 핵심 기능 요구사항
**① OCR 기반 스테이지 자동 생성**
- 사용자가 단어장 표지와 내지를 카메라로 촬영하여 업로드.
- 표지 텍스트(예: Insight Link 5-6)를 분석해 '대분류(책/대륙)' 생성.
- 내지 텍스트(예: Unit 5)와 단어 표(영어-한글)를 분석해 '중분류(스테이지)' 및 '단어 리스트' 생성.

**② 하이브리드 AI 시각 연상 시스템 (비용 최적화)**
- 단어의 뜻을 직관적으로 보여주는 AI 이미지 제공.
- **이미지 호출 로직 (비용 절감 핵심):**
  1. 단어 스캔 시 자체 이미지 DB 우선 조회.
  2. DB에 이미지가 존재하면 해당 URL 반환 (캐싱 활용).
  3. DB에 없을 경우에만 AI 이미지 생성 모델 호출 -> 이미지 생성 후 DB에 저장 및 반환.

**③ 게이미피케이션 학습 모드**
- **보스전:** 외워야 할 단어를 몬스터로 상정. 정답을 맞출 때마다 타격 이펙트와 함께 몬스터 HP 감소.
- **리벤지 매치 (오답 노트):** 틀린 단어들은 '도망친 몬스터'로 분류되어, 빠른 속도감의 미니 게임(예: 두더지 잡기) 형태로 재도전 유도.
- **보상 시스템:** 스테이지 클리어 시 골드/경험치 지급.

## 3. 데이터베이스 스키마 (초안)
- **Book (대분류):** ID, Title (책 이름), Cover_Image_URL
- **Unit (중분류):** ID, Book_ID, Title (단원 이름), Status (클리어 여부)
- **Word (단어):** ID, Unit_ID, English_Word, Korean_Meaning
- **Image_Cache (시각 연상):** ID, English_Word (Unique Key), Image_URL
