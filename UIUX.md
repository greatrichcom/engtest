# UI/UX 설계서: Zen-Pop (모바일/태블릿 최적화)

## 1. 디자인 컨셉
- **테마:** 둥글둥글하고 친근한 3D 아트워크, 파스텔 톤의 힐링 컬러와 경쾌한 팝 컬러의 조합.
- **조작:** 텍스트 타이핑(키보드)을 최소화하고, 드래그(Drag), 드롭(Drop), 스와이프(Swipe), 선 긋기 등 모바일 터치 인터랙션을 극대화.

## 2. 메인 화면 (Lobby / Base Camp)
- **상단바 (Status):** 좌측엔 사용자의 귀여운 아바타와 레벨 표시. 우측엔 보유 재화(골드, 보석).
- **중앙 (모험 지도):** - 등록된 단어장(대분류)들이 동화책이나 마법의 탑 형태로 입체적으로 배치됨.
  - 책을 선택하면 하위 Unit(스테이지)들이 노출됨. 클리어한 Unit은 '별 3개', 미클리어는 '자물쇠' 아이콘 표시.
- **하단 (Action):** 중앙에 눈에 띄는 큰 플로팅 버튼(FAB)으로 "📷 단어장 스캔하여 던전 만들기" 기능 배치. 좌우로 '상점', '몬스터 도감' 메뉴 배치.

## 3. 게임 화면 (Battle UI) - 화면 분할 구조
- **상단 영역 (적 & 힌트):**
  - 화면 중앙에 물리쳐야 할 몬스터 배치 (상단에 HP 게이지 바).
  - 몬스터 옆 말풍선 형태로 'AI가 생성한 해당 단어의 웃긴 연상 이미지' 노출.
  - 우측 상단에 현재 스테이지 진행도(예: 3/30 마리) 미니맵 표시.
- **하단 영역 (인터랙티브 조작 패널):**
  - **타입 A (스펠링 맞추기):** 빈칸( _ _ _ _ ) 아래에 알파벳 구슬들이 무작위로 흩어져 있음. 구슬을 빈칸으로 드래그 앤 드롭.
  - **타입 B (뜻 맞추기):** 좌측에 영어, 우측에 한글 뜻이 방울 형태로 배치됨. 손가락으로 드래그해서 슬라임처럼 선을 이어 정답 연결.
- **피드백 이펙트:** 정답 시 화면 진동(햅틱), 화려한 타격 이펙트, 몬스터 피격 애니메이션 적용.

## 4. 결과 및 리벤지 매치 화면
- **결과 화면:** 스테이지 종료 시 보물상자가 등장하며 화면 연타로 보상(골드) 획득 연출.
- **리벤지 화면:** 본 게임에서 틀린 단어들을 감옥에 갇히지 않은 실루엣 몬스터로 표현. "리벤지 매치" 버튼 클릭 시 오답 복습용 두더지 잡기 미니게임으로 전환.
